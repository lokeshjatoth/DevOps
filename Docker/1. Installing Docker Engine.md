Single Command For Installing Docker Environment
```
sudo apt-get update && sudo apt-get install -y docker.io && sudo systemctl start docker &&  
sudo systemctl enable docker && sudo docker --version
```

1. Update the APT package index.
```
sudo apt-get update -y
```

2. Install certificate and download tools.
```
sudo apt-get install ca-certificates curl -y
```

3. Create the GPG keyring directory.
```
sudo install -m 0755 -d /etc/apt/keyrings
```

4. Download Docker’s GPG key.
```
sudo curl -fsSL https://download.docker.com/linux/ubuntu/gpg -o /etc/apt/keyrings/docker.asc
```

5. Make the GPG key readable by all users.
```
sudo chmod a+r /etc/apt/keyrings/docker.asc
```

6. Add Docker’s repository to APT sources.
```
echo \
  "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.asc] https://download.docker.com/linux/ubuntu \
  $(. /etc/os-release && echo "${UBUNTU_CODENAME:-$VERSION_CODENAME}") stable" | \
  sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
```

7. Update apt index again after adding Docker repo.
```
sudo apt-get update -y
```

8. Install Docker Engine and its components.
```
sudo apt-get install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin -y
```

9. Start the Docker daemon.
```
sudo systemctl start docker
```

10. Enable Docker to start automatically on boot.
```
sudo systemctl enable docker
```

11. Verify Docker works by running hello-world container.
```
sudo docker run hello-world
```

12. Add your user to the docker group (optional step).
```
sudo usermod -aG docker $USER
```

13. List all Docker images to confirm `hello-world` image was downloaded.
```
sudo docker images
```

14. List all containers including stopped ones.
```
sudo docker ps -a
```
